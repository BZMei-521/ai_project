# 分镜动画应用 PRD v1

## 1. 文档信息
- 产品代号：Seko-like Storyboard Pro
- 版本：v1.0（MVP）
- 日期：2026-02-25
- 目标：交付可上线的专业分镜动画单机版（桌面端）

## 2. 产品目标
### 2.1 业务目标
1. 支持动画/广告/短片团队从“分镜绘制”到“Animatic 导出”的完整闭环。
2. 将单镜头制作时间控制在 5 分钟内（含绘制、排序、预览）。
3. 第一版聚焦专业创作效率，不做社区和复杂云协作。

### 2.2 用户目标
1. 快速创建镜头并在时间轴中调整节奏。
2. 在单一界面完成绘制、标注、对白、时长设定。
3. 一键导出 PDF 分镜稿和 MP4 animatic 用于评审。

## 3. 用户画像
1. 导演/分镜师：高频使用时间轴、镜头排序、批注。
2. 动画师：关注镜头内容、镜头衔接、动作节奏。
3. 制片/PM：关注导出成果、版本可追溯、评审效率。

## 4. MVP 范围
### 4.1 In Scope
1. 项目管理：创建/打开/保存本地项目。
2. 镜头卡片：新建、删除、复制、拖拽排序、时长设置。
3. 分镜画布：图层、画笔、橡皮、变换、洋葱皮。
4. 时间轴：按镜头排列、支持镜头时长编辑和播放头预览。
5. Animatic 预览：按时间轴播放镜头，支持单音轨。
6. 资产库：角色/场景/道具作为可复用素材。
7. 导出：PDF（分镜版式）+ MP4（animatic）。
8. 自动保存与崩溃恢复。

### 4.2 Out of Scope（v1 不做）
1. 实时多人协同编辑。
2. 移动端和 Web 在线编辑。
3. 高级特效合成（粒子、3D 相机、复杂转场）。

## 5. 核心用户流程
1. 创建项目 -> 添加序列 -> 新建镜头。
2. 在画布绘制分镜并填写对白/备注。
3. 拖拽镜头调整顺序与时长。
4. 导入临时音轨并预览 animatic。
5. 导出 PDF 给导演评审，导出 MP4 给剪辑沟通。

## 6. 功能需求
### 6.1 项目与文件
1. 支持 `.sbproj` 项目文件（JSON 元数据）与素材目录结构。
2. 支持最近项目列表。
3. 支持手动保存和自动保存（默认 30 秒）。

### 6.2 镜头卡片
1. 卡片字段：镜头号、标题、时长（帧）、对白、备注、标签。
2. 支持批量选择和批量移动。
3. 支持按标签筛选镜头。

### 6.3 分镜画布
1. 图层：新增、删除、重命名、显隐、锁定。
2. 画笔：大小、不透明度、颜色、压感（可选）。
3. 操作：撤销/重做（至少 100 步）。
4. 洋葱皮：前后各 1-3 帧可配置。

### 6.4 时间轴与预览
1. 时间轴最小单位：帧；支持 12/24fps。
2. 播放控制：播放/暂停、跳转上一镜头/下一镜头。
3. 音轨：导入 wav/mp3 并对齐开始时间。

### 6.5 资产库
1. 分类：角色、场景、道具。
2. 支持拖拽素材到画布。
3. 支持素材引用关系（镜头使用记录）。

### 6.6 导出
1. PDF：支持镜头网格、字段显示开关（对白/备注）。
2. MP4：按时间轴渲染，支持音轨混入。
3. 导出预设：1080p，24fps，H.264。

## 7. 非功能需求
1. 性能：200 镜头项目打开时间 < 5 秒。
2. 稳定性：异常退出后可恢复最近自动保存版本。
3. 可用性：核心操作有快捷键；首次上手 < 30 分钟。
4. 平台：macOS / Windows（Tauri 桌面端）。

## 8. 指标（MVP 验收）
1. 创建 100 镜头项目并完整编辑无崩溃。
2. 单次导出 10 分钟 animatic 成功率 >= 99%。
3. 用户测试中“完成一个 30 镜头样片”时长 <= 2 小时。

## 9. 里程碑
1. M1（第 1-2 周）：项目结构、数据模型、基础 UI 框架。
2. M2（第 3-5 周）：镜头卡片、画布基础、时间轴基础。
3. M3（第 6-8 周）：预览、音轨、导出（PDF/MP4）。
4. M4（第 9-10 周）：稳定性优化、自动恢复、Beta 验收。

## 10. 风险与应对
1. 时间轴模型返工风险：先冻结数据结构并写迁移机制。
2. 导出性能风险：采用离屏渲染 + 分段编码。
3. 画布交互复杂风险：先保证笔刷/图层稳定，再扩展高级功能。
