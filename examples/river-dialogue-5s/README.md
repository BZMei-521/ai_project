# 河边对话 5 秒示例（两人）

## 文件
- `river_dialogue_5s_project_backup.json`：可直接在主界面「导入备份」加载的完整示例项目
- `river_dialogue_5s_shot_script.json`：可在 AI 流水线「已分镜脚本(JSON)」直接粘贴导入的镜头脚本

## 快速生成步骤
1. 打开项目后，先点顶部「导入备份」，选择 `river_dialogue_5s_project_backup.json`
2. 打开右侧「AI 生成流水线」
3. 确认 ComfyUI 已连接（状态显示“已连接”）
4. 在「已分镜脚本(JSON)」粘贴 `river_dialogue_5s_shot_script.json` 内容，点击「导入镜头脚本」
5. 点击「一键生成整片」

## 规格
- 总时长：5 秒
- FPS：24
- 总帧数：120 帧（4 镜头 x 30 帧）
- 场景：河边对话
